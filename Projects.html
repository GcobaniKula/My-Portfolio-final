Transportation Website

Description

The Transportation Website is an informative website, created with the aim of allowing the user to be informed in terms of transportation prices and the time to reach a destination. Image relocating to a new city and the next day you have an interview, but you do not know the price and time it takes to reach the place you are going to be interviewed in, the transportation website addresses that exactly! To reach a particular place the prices differ based on the time and the type of transportation you are using, example in Western Cape if you are traveling during peak up hours the price of a bus will be more compared to traveling at a normal time so that information will be addressed by the website we created.

Programming Language

  • HTML for frontend
  • CSS for frontend
  • JavaScript for backtend/userinterface
